# Portfolio Assets

This folder contains image assets for the portfolio website.

## Required Images

Add the following images to customize your portfolio:

- `profile.png` - Your profile photo (recommended: 400x400px)
- `nutripal_screenshot.png` - Screenshot of NutriPal app
- Project thumbnails for the gallery

## Image Formats

Supported formats: PNG, JPG, WebP
Recommended: Use WebP for better compression
